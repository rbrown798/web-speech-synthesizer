# Web Speech Synthesizer

A parallel formant speech synthesizer with a text-to-speech interface made using the Web Audio API. 

# Credits

This project includes a modified version of the CMU Pronouncing Dictionary (CMUdict).
Copyright (C) Carnegie Mellon University.
See `CMUDICT-LICENSE.txt` for license information. 

Formant measurements for vowels, approximants, and nasals were adapted from Dennis H. Klatt, 
"Software for a Cascade/Parallel Formant Synthesizer" (1980). 
Fricative parameters were developed independently.
